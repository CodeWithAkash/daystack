# DayStack

DayStack is a modern productivity tracking platform designed to help users maintain discipline, track daily habits, and build consistency.

## Features

- Daily habit tracker
- Streak system
- Pomodoro productivity timer
- Motivational quotes
- Habit heatmap visualization
- AI productivity suggestions
- Mobile and web support

## Tech Stack

Frontend
- React
- Vite
- Chart.js
- Framer Motion

Backend
- Flask
- Python
- MongoDB

Mobile
- React Native
